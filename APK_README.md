## 🚀 First-Time Setup

### Create Your Account
1. Launch the app
2. Tap **"Sign Up"**
3. Enter your details:
   - Name (e.g., "John Smith")
   - Username (e.g., "john.inspector")
   - Password (minimum 6 characters)
   - Inspector ID (e.g., "INS-001")
4. Tap **"Sign Up"**
5. You're in! Dashboard will load automatically

### Your First Inspection
1. Tap **"New Assessment"** on dashboard
2. Enter location (e.g., "CH 165.2, LHS")
3. Tap camera icon and take a photo
4. Select defect code (e.g., tap "FC1" for fine crack)
5. Select impact category (tap "Moderate")
6. Choose preset comment or type custom text
7. Tap **"Save to Worksheet"**
8. Success! Item saved to database

### Export Your Report
1. Return to dashboard
2. Tap **"Export Reports"**
3. Choose **"Export as PDF"** or **"Export as Excel"**
4. File saved to Downloads folder
5. Tap **"Share"** to send via email/WhatsApp
6. Done!

## 🔧 Troubleshooting

### ❓ "App not installed" error
- **Fix:** Uninstall any previous version first
- Check you have 150+ MB free storage
- Enable "Unknown sources" in Settings

### ❓ Camera not working
- **Fix:** Go to Settings → Apps → Dilapidation Survey → Permissions
- Enable **Camera** permission
- Restart the app

### ❓ Can't find exported files
- **Fix:** Open your file manager app
- Navigate to **Downloads** folder
- Sort by "Date modified" (newest first)
- Look for files starting with "Dilapidation_Report_"

### ❓ Login fails after signup
- **Fix:** Password is case-sensitive
- Must be at least 6 characters
- Try signing up with a different username

### ❓ App crashes on startup
- **Fix:** Uninstall app completely
- Clear device cache (Settings → Storage)
- Reinstall APK
- If problem persists, note your Android version and report issue

## 💾 Data Management

### Where is my data stored?
- **Database:** `/data/data/com.fieldlens.dilapidation/databases/`
- **Photos:** App's private cache directory (compressed JPEGs)
- **Exports:** `/storage/emulated/0/Download/` (public Downloads folder)

### How do I backup my data?
- **Regular exports:** Generate PDF/Excel reports regularly
- **Cloud backup:** Upload exported files to Google Drive, Dropbox, etc.
- **Manual backup:** Use ADB to pull database file (advanced users)

### What happens if I uninstall?
- ⚠️ **All data is permanently deleted**
- Export important reports before uninstalling
- No cloud recovery available (offline-only app)
